3. What are self-closing tags?

Self-closing tagts are type of tags, there is no need to close the tag.
Example: break tag, image tag etc..

4. How many types of headings are there? How to decide to use where?

There are six types of headings are there.

  • From h1 to h6
  • h1 tag is used to add title of the page and remaining tags are used for sub-headings and statements
  • h1 tag have higher priority where as h6 tag have least priority

7. What are attributes?

  • It is a special keyword, which is used as modifier in html elements
  • It is used to provide additional information to an element
  • Attributes should be written in case-sensitive
  • It adjusts the behaviour of an html element like color, size etc.

8. Difference between HTML tags and Elements

HTML tags Elements
Tags are the starting and ending parts of an HTML element Elements enclose the contents in between the tags.
They begin with < symbol and end with > symbol. Whatever written inside < and > are called tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag.
